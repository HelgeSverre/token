## About

Token is a lightweight, high-performance text editor built with Rust. It is designed to be a simple, distraction-free environment for writing beautiful and efficient code.

## Features

*   **Fast & Lightweight:** Built in Rust for native performance.
*   **Minimalist UI:** A clean interface that puts your code first.
*   **Syntax Highlighting:** Clear and effective highlighting.
*   **Cross-Platform:** Runs on macOS, Windows, and Linux.

## Building from Source

To build Token from source, you will need the Rust toolchain installed.

```bash
git clone <repository-url>
cd token
cargo build --release
```
