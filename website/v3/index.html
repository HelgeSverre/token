01

Multi-cursor editing

Add cursors anywhere with Option+Click. Add cursors above/below with Option+Option+Arrow. All cursors move and edit simultaneously. Overlapping cursors merge automatically.

⌥+Click add cursor ⌘+J select next match
02

Split views

Horizontal and vertical splits with independent viewports. Same file, different positions. Different files, same window.

03

Tree-sitter syntax

Incremental parsing with real AST understanding. Not regex. Accurate highlighting that actually works.

04

CSV spreadsheet

View CSV, TSV, PSV as a proper table. Navigate with arrow keys. Edit cells in place. Auto-sizing columns.

05

YAML config

Keymaps and themes in plain YAML. No JSON schema hunting. No settings UI maze. Just text.

Get Token

Pick your platform. Extract. Run. That's it.

macOS Apple Silicon .tar.gz macOS Intel .tar.gz Linux x86_64 .tar.gz Windows x86_64 .zip

Or install with one command

curl -L https://github.com/HelgeSverre/token/releases/latest/download/token-macos-aarch64.tar.gz | tar xz && sudo mv token /usr/local/bin/
curl -L https://github.com/HelgeSverre/token/releases/latest/download/token-linux-x86_64.tar.gz | tar xz && sudo mv token /usr/local/bin/
git clone https://github.com/HelgeSverre/token && cd token && make release && sudo mv target/release/token /usr/local/bin/
