Switching themes

Use the Command Palette to switch themes:

1 Open Command Palette with ⌘⇧A
2 Type "Switch Theme"
3 Select your theme

Creating custom themes

Themes are YAML files with a structured format. Create yours in the themes directory:

~/.config/token-editor/themes/my-theme.yaml 
version: 1
name: "My Theme"
author: "Your Name"
description: "My custom Token theme"

ui:
  editor:
    background: "#282C34"
    foreground: "#ABB2BF"
    current_line_background: "#2C313A"
    cursor_color: "#528BFF"
    selection_background: "#3E4451"
    secondary_cursor_color: "#528BFF80"

  gutter:
    background: "#282C34"
    foreground: "#4B5263"
    foreground_active: "#ABB2BF"
    border_color: "#3E4451"

  status_bar:
    background: "#21252B"
    foreground: "#9DA5B4"

  syntax:
    keyword: "#C678DD"
    function: "#61AFEF"
    string: "#98C379"
    number: "#D19A66"
    comment: "#5C6370"
    type: "#E5C07B"
    variable: "#E06C75"
    operator: "#56B6C2"

Color reference

All available theme properties organized by section.

ui.editor Main editor area

background Editor background color
foreground Default text color
current_line_background Background of the current line
cursor_color Primary cursor color
selection_background Selected text background
secondary_cursor_color Multi-cursor secondary cursor

ui.gutter Line numbers

background Gutter background
foreground Inactive line numbers
foreground_active Current line number
border_color Border between gutter and editor

ui.sidebar File tree

background Sidebar background
foreground Default text color
selection_background Selected item background
hover_background Hovered item background
folder_icon Folder icon color
file_icon File icon color

ui.syntax Syntax highlighting

keyword if, else, fn, let
function println, map, filter
string "hello world"
number 42, 3.14, 0xFF
comment // comment
type String, Vec, Option
variable count, name, result
operator +, -, =>, ::
constant TRUE, NULL, PI
tag <div>, <span>
attribute class=, id=
escape \n, \t, \x00

Tips

  • Start from an existing theme — Copy one of the built-in themes and modify the colors
  • Use hex colors — 6-digit (#RRGGBB) or 8-digit with alpha (#RRGGBBAA)
  • Test with different file types — Verify syntax colors across multiple languages
  • Consider contrast — Ensure sufficient contrast for readability
